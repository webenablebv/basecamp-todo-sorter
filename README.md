# Basecamp to-do sorter
Chrome extension to sort to-do's by due date

## Installation instructions
- Download repository code as zip and unzip
- Open Chrome Extensions Manager
- Enable _developer mode_
- Click **Load unpacked** and select the folder with the Chrome extension files

The extension will add a button to the options menu of a to-do list.
